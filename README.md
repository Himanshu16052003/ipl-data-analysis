# IPl-data-analysis
# 🏏 IPL Data Analysis using Python

## 📌 Project Overview

This project analyzes Indian Premier League (IPL) match data using Python. It explores team performance, player statistics, batting and bowling records, and match trends using data visualization.

## 📊 Features

- Team-wise match wins
- Orange Cap analysis
- Purple Cap analysis
- Most sixes and fours
- Toss analysis
- Venue analysis
- Winning percentage
- Player performance
- Interactive visualizations

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset

- matches.csv
- deliveries.csv
